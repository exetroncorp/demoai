Design and create a virtual drum kit simulation. The simulation must feature 3D graphics or photorealistic 2D assets mimicking the aesthetic of Logic Pro X's Drum Kit Designer. It must feature the following items: A realistic simulation of a standard drum kit that can be played in real-time using the computer keyboard. A full set of selectable drum components available for the user to "play," including: Kick Drum, Snare, Closed Hi-Hat, Open Hi-Hat, High Tom, Floor Tom, Crash Cymbal, and Ride Cymbal. The simulation should realistically animate the instruments when triggered (e.g., the drum skins vibrating or the cymbals swaying) and include visual feedback (highlighting) to show which key was pressed. The simulation should include a "Key Map" overlay that shows the user which keyboard keys correspond to which drum (e.g., Spacebar for Kick, 'F' and 'J' for Snare, etc.). A focus on visual realism and low-latency audio response is paramount.

There also must be an "AUTOPLAY" feature, with 4 selectable drum tracks for the kit to auto play. You may use any library for this implementation (such as Three.js for graphics or Tone.js for audio), but it must be contained within a single HTML file/script, and be able to be opened and played in the Chrome browser.
